# Rogue Command Core Gameplay Loop: How One Battle Becomes a Build

Rogue Command is not about memorizing one perfect build order. Each run asks you to turn changing rewards, RTS execution, and long-term unlocks into a build that can survive the next fight.

Understanding this loop is more useful than memorizing a single unit or reward.

## The Run Starts With Three Axes

Before a run gets moving, your choices are shaped by three axes: Engineer, Economy, and Specialist. The Engineer defines how you build and operate on the map. The economy choice affects how quickly Crystal becomes production. The Specialist gives your early army its first combat identity.

These choices are connected. A defensive Specialist can support slower expansion. A mobile Specialist may help you claim space faster. A fragile economy needs protection. A slower economy needs cleaner production choices.

## Battles Are Still RTS Battles

Every map asks you to perform classic RTS tasks: gather resources, build a base, produce units, scout, defend, and attack. Roguelite rewards do not replace those fundamentals.

Before pushing forward, check whether your economy is working, your production is active, and your army can survive the next contact. If one of those pillars is weak, fix the rhythm before expanding the fight.

## Rewards Turn Momentum Into Direction

After battles, rewards such as Blueprints, Upgrades, and Hacks push your run in new directions. Blueprints add new build options. Upgrades improve existing tools. Hacks change broader rules or interactions.

Do not ask only whether a reward is strong. Ask what it lets you do in the next battle. Does it improve your current army? Does it cover a weakness? Does it open a mechanic you can already support?

## Long-Term Growth Matters

Rogue Command also uses long-term progression. Archive tokens and the Battle Archive help future runs start from a better position or gain more control over randomness.

This means a failed run can still be useful. The best review question is simple: what was the earliest point where the run became unstable? That answer should shape your next Battle Archive investment and your next run goal.

## The Core Lesson

The loop is: choose a starting identity, execute the RTS battle, pick rewards that serve the current build, then convert lessons into long-term growth. Strong builds are not found all at once. They are assembled one useful decision at a time.
